# vue3-spa

默认模版为vue3，[vue2模版](https://github.com/asasugar/vue-spa/tree/VUE2_SPA_TAG_V1)，[vue2移动端模版](https://github.com/asasugar/vue-spa/tree/vue-spa-mobile)

## 项目运行
#### 推荐使用yarn
```
npm i -g yarn

yarn config set registry http://registry.npm.taobao.org/
```

#### 安装依赖

```
yarn
```

#### 项目运行

```
yarn dev
```
#### 项目打包

```
yarn build
```
